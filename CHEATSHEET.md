# PEN Cheatsheet (v1.0.0)

## Structure (S)
- `S2`: Couplet
- `S3`: Tercet
- `S4`: Quatrain
- `S~`: Free verse

## Meter (M)
- `M=`: Regular
- `M≈`: Loose
- `M~`: Free

## Voice (V)
- `V^`: Imperative ("Come", "Let")
- `V!`: Apostrophe ("O ...")
- `V.`: Indicative
- `V?`: Interrogative

## Rhetoric (R)
- `R=`: Parallelism/Anaphora
- `R≠`: Antithesis
- `R:`: Metaphor/Simile
- `R+`: Accumulation

## Domain (D)
- `D*`: Divine/Cosmic
- `D.`: Somatic/Body
- `D~`: Nature/Elemental
- `D_`: Abstract/Mental
- `D@`: Domestic/Social
- `>` : Shift (e.g., `D_>D.`)

## Vector (T)
- `T↑`: Ascending
- `T↓`: Descending
- `T↔`: Neutral
- `T→`: Forward

## Live Mode (Shorthand)
- Omit prefixes if standard order is used.
- `>`: Domain shift
- Example: `4 ≈ ^ ≠ *`
